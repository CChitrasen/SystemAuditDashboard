About System Audit Dashboard

System Audit Dashboard is a centralized Linux system monitoring tool developed in Python using Tkinter for the GUI. It gathers key system metrics—including kernel version, operating system details, logged-in user, process and port counts, load averages, available memory, and disk usage—and displays them in real time.

Features

  • Real-Time Monitoring: Displays live data, updated periodically.
  • Threaded Data Collection: Gathers system metrics in a separate thread to keep the GUI responsive.
  • Configurable Update Interval: Set your preferred refresh rate using the dashboard settings.
  • Interactive Load Average Graph: Visualize load average values for 1, 5, and 15 minutes with a dynamic bar chart.
  • Comprehensive Metrics: Shows kernel version, OS details, logged-in user, processes, open ports, memory, and disk usage.
  • Advanced Logging: All operations and errors are logged to sys_audit_dashboard.log for troubleshooting.
